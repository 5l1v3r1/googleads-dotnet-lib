Detailed Description

Proposal line items are created from products, from which their properties are copied.

Targeting Google.Api.Ads.AdManager.v201711.Product.builtInTargeting
getset

The targeting for the created ProposalLineItem. For those scenarios using sales management, it's a combination of ProductTemplate::productSegmentation and ProductTemplate::builtInTargeting. See details in ProductTemplate. This attribute is read-only when:

  • using programmatic guaranteed, using sales management.
  • not using programmatic, using sales management.
