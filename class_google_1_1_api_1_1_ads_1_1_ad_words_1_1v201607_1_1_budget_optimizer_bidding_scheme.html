Detailed Description

In budget optimizer, Google automatically places bids for the user based on their daily/monthly budget.

Note: This bidding strategy has been deprecated and replaced with TargetSpend. We no longer allow advertisers to opt into this strategy– solely exists so that advertisers can access campaigns that had specified this strategy.

This is disabled for AdX.
