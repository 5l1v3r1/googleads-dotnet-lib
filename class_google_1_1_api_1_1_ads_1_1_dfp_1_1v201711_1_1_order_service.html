Detailed Description

Provides methods for creating, updating and retrieving Order objects.

An order is a grouping of LineItem objects. Line items have a one-to-one relationship with orders. Each line item can belong to only one order. Orders, however, have a one-to-many relationship with line items, meaning each order can have multiple line items. An order can be used to manage the line items it contains.
