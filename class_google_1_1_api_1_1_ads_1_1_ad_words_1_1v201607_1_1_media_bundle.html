Entry in the ZIP archive used to display the

MediaBundle in an Ad. This field can only be set and returned when the MediaBundle is used with the AdGroupAdService. If this field is set when calling MediaService, an error will be returned.

To use a MediaBundle that was created with the MediaService in an Ad, create a bundle and set the mediaId and entryPoint fields.
