Criterion [] Google.Api.Ads.AdWords.v201601.BudgetSuggestionSelector.criteria
getset

Criteria for budget suggestion. The following criteria are supported:

  • ProductService
  • Language
  • Location
  • Proximity

Please note that Address is not supported in Proximity. Please use GeoPoint instead. ProductService, Language, and at least one of Location and Proximity must be given, otherwise default suggestion will be returned. This field must contain distinct elements. This field must not contain elements.
