Detailed Description

This code example creates an experiment using a query percentage of 10, which defines what fraction of auctions should go to the control split (90%) vs. the experiment split (10%), then adds experimental bid changes for criteria and ad groups. To get campaigns, run GetCampaigns.cs. To get ad groups, run GetAdGroups.cs. To get criteria, run GetKeywords.cs.
