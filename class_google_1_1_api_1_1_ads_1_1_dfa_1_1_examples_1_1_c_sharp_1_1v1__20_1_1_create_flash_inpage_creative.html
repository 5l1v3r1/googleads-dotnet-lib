Detailed Description

This code example creates a "Flash InPage" creative in a given advertiser or campaign. If no campaign is specified then the creative is created in the advertiser provided. To get assets file names, run CreateHTMLAsset.cs and CreateImageAsset.cs. To get a size id, run GetSize.cs. To get a creative type id, run GetCreativeType.cs.
