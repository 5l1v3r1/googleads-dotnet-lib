DateRange Google.Api.Ads.AdWords.v201802.Selector.dateRange
getset

Range of dates for which you want to include data. If this value is omitted, results for all dates are returned.

Note: This field is only used by the report download service. For all other services, it is ignored.

This range must be contained within the range [19700101, 20380101].
