string Google.Api.Ads.Common.Lib.AdsSoapClient.GetDefaultNamespace ( )
inlineprotected

Gets the default XML namespace, based on the type of this object.

Returns
The XML namespace to which this object is serialized, or an empty string if the method fails to retrieve the default namespace.

All the services making use of the XML Serialization framework (including ones generated by wsdl.exe and xsd.exe) will have a WebServiceBindingAttribute decoration, something like:

[System.Web.Services.WebServiceBindingAttribute( Name = "SomeServiceSoapBinding", Namespace = "https://the_xml_namespace_for_serializing")] public partial class SomeService : SoapHttpClientProtocol { ... }

The only exception to this rule is when we choose to write our own serialization framework, by implementing IXmlSerializable on AdsSoapClient. If and when someone does that, and someone were to call this method, then he/she will get an empty string from this method.
