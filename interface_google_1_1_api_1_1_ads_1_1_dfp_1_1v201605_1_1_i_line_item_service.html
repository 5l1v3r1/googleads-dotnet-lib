Detailed Description

Provides methods for creating, updating and retrieving LineItem objects.

Line items define the campaign. For example, line items define:

  • a budget
  • a span of time to run
  • ad unit targeting

In short, line items connect all of the elements of an ad campaign.

Line items and creatives can be associated with each other through LineItemCreativeAssociation objects. An ad unit will host a creative through both this association and the LineItem::targeting to it.
