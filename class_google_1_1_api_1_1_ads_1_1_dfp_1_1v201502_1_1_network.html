long Google.Api.Ads.Dfp.v201502.Network.contentBrowseCustomTargetingKeyId
getset

The ID of the

CustomTargetingKey that is used to categorize video content in DFP. Values of this CustomTargetingKey can be used as part of the Targeting::contentTargeting for targeting line items to video categories. Additionally, you can use the values of this key with the ContentService to search for all that are associated to a particular key/value combination.
