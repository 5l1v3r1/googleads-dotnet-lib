Detailed Description

A Feed identifies a source of data and its schema. The data for the Feed can either be user-entered via the FeedItemService or system-generated, in which case the data is provided automatically.

FeedAttribute [] Google.Api.Ads.AdWords.v201603.Feed.attributes
getset

The Feed's schema. In SET operations, these attributes will be considered new attributes and will be appended to the existing list of attributes unless this list is an exact copy of the existing list (as would be obtained via FeedService::get). If an empty attributes list is provided, the existing list of attributes will not be changed. This field can be selected using the value "Attributes". This field must not contain

elements. This field is required and should not be null when it is contained within Operators : ADD, SET.
