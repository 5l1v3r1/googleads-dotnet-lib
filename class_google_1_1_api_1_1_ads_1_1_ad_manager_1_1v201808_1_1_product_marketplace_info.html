string Google.Api.Ads.AdManager.v201808.ProductMarketplaceInfo.additionalTerms
getset

Additional terms shown to the buyer in Marketplace. When using sales management, this attribute is populated with the additional terms value from the product template this product is created from. To overwrite this, set the #additionalTermsSource to ValueSourceType::DIRECTLY_SPECIFIED when setting the value of this field.

ValueSourceType Google.Api.Ads.AdManager.v201808.ProductMarketplaceInfo.additionalTermsSource
